Element

ウェブコンポーネント
要素 説明
<template>
  • ページが読み込まれたとき：すぐにレンダリングされない。
  • 実行時：JavaScript を使って後からインスタンス化することができる
  • HTMLを保持するためのメカニズム
<slot>
  • ウェブコンポーネント 技術群の一部
  • この要素は、ウェブコンポーネント内のプレースホルダーで、自分自身でマークアップを埋めることができ、別個の DOM ツリーを作成して一緒に表示することができる。
<span slot="desc">挿入される概要</span>
<template>
  <slot name="desc">defo DESCRIPTION</slot>
</template>
ウェブコンポーネントは要するに、独自の要素を作成して通常の HTML 要素であるかのように使用できる、HTML 関連技術です。

try


      <template>
        this template is not shadow
      </template>
    


      <template shadowrootmode="open">
        this template is shadow
      </template>

shadowrootmode属性を渡すと既存HTMLが隠れてしまう..
