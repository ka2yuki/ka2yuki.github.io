```sh
gh-pages -d build -b main
```
- `gh-pages`：コマンド.
- `-d`：プッシュするディレクトリ.
- `-b`：プシュ先のリモートブランチ.
  - まだ作成していないブランチでも、新規作成できる.

gh-pages [install](https://www.npmjs.com/package/gh-pages).
